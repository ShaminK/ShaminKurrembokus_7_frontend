Projet 7 d'Openclassrooms : Front-End

Pour faire fonctionner le projet 7 vous devez:
- avoir installer NodeJs
- cloner le dépôt git
- ouvrir votre terminal
- aller dans votre dossier 'frontend' : cd frontend
- puis sur votre terminal: npm start
- telecharger la partie backend sur ce lien puis suivre les instructions du README :
https://github.com/ShaminK/ShaminKurreembokus_7_backend
